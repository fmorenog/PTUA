## LAB 02-1 : BASIC TOOLS

**What does ls do?**

Short for listing. Lists the contents of the current directory.

**What does `ls -al` do?**

Opposite of `ls`, lists all contents **not** in the current directory.

**What does `pwd` do?**

Shows you the directory you're currently working on

**What does `mkdir PTUA` do?**

Creates a folder named PTUA inside the directory I'm working on.

**What does `cd PTUA` do?**

changes my directory to PTUA

**Enter into your desktop folder by using `cd Desktop`**

```r
cd ~/Desktop
```

**Create a directory called `PTUA-lab2`**

Creates a folder called PTUA-lab2 on my desktop

```r
mkdir PTUA-lab2
```

**`cd PTUA-lab2`**

Changes my directory to work inside `PTUA-lab2`

**What is your current working directory now?**

Currently I'm working inside `PTUA-lab2`

```r
(base) fabianmorenogarza@glaroam2-148-107 PTUA-lab2 % pwd
```
